Welcome to the Require Script Wiki

About the Wiki

This wiki is dedicated to the Roblox require script community. It explores the history, hubs, scripts, and archives, highlighting some of the most influential projects and contributors. Here you can learn about BTT Console, BTT Chat Hax, ExSer Archives, and other notable require scripts.

BTT Console

BTT Console is the most widely used Roblox require script hub ever, created by Bacon The Thing (BTT). It allowed users to load a suite of scripts via require modules in the Roblox require script community.

  • GUI-style hub interface
  • Collection of popular scripts and modules
  • Highly referenced in require script archives

BTT Chat Hax

BTT Chat Hax was a chat script in the style of BTT Console. It stopped working after Roblox updated the chat system and after BTT left the require script community.

  • Enabled advanced chat manipulation
  • Built in the style of BTT Console
  • Historical significance in the require script community

ExSer Archives

ExSer is known for archiving BTT's scripts and other require script modules after authors left or assets were deleted. Many historic modules, like BTT Console backups and other require script tools, can still be found in these archives.

The archives also preserve old hub layouts, module lists, and key information for research and documentation purposes.
