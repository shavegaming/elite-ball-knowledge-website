BTT Console

BTT Console is the most used Roblox require script hub ever, created by Bacon The Thing (BTT). It allowed users to load a suite of scripts via require modules in the Roblox require script community.

  • GUI-style hub interface
  • Collection of popular scripts and modules
  • Highly referenced in require script archives

BTT Chat Hax

BTT Chat Hax was a chat script in the style of BTT Console. It stopped working after Roblox updated the chat system and after BTT left the require script community.

ExSer Archives

ExSer is known for archiving BTT's scripts and other require script modules after authors left or assets were deleted. Many historic modules can still be found in these archives.
